ListItemAvatar
==============

`<ListItemAvatar>` is a simple wrapper to apply the `dense` mode styles to `Avatar`.

```
<ListItemAvatar>
  <Avatar>
</ListItemAvatar>
```

|:-----|:-----|:--------|:------------|
