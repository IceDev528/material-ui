関連する [Toggle buttons](https://material.io/design/components/buttons.html#toggle-button)のグループを強調するには、次の操作を行います。 グループは共通のコンテナーを共用する必要があります。

`ToggleButtonGroup`は、次の場合に子ボタンの選択を制御します。 独自の`value`propが与えられます。
