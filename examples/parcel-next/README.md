# Parcel v4-beta example

## How to use

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

```sh
curl https://codeload.github.com/mui-org/material-ui/tar.gz/next | tar -xz --strip=2  material-ui-next/examples/parcel-next
cd parcel-next
```

Install it and run:

```sh
npm install
npm run start
```

## The idea behind the example

[Parcel](https://github.com/parcel-bundler/parcel) is a Blazing fast, zero configuration web application bundler.
