# Joy UI - Create React App example in TypeScript

`@mui/joy` is still in early development and is not recommended for use in production.

## How to use

Download the example [or clone the repo](https://github.com/mui/material-ui):

<!-- #default-branch-switch -->

```sh
curl https://codeload.github.com/mui/material-ui/tar.gz/master | tar -xz --strip=2 material-ui-master/examples/joy-cra-ts
cd joy-cra-ts
```

Install it and run:

```sh
npm install
npm start
```
